# The 86 Club Workspace V4.5

Workspace privado para The 86 Club con Firebase, Firestore, navegación por categorías, etapas guiadas, tareas accionables, perfiles, disponibilidad semanal y calendario semanal por perfil.

## V4.5

Agrega calendario semanal dentro de cada perfil:

- Vista de lunes a domingo según disponibilidad configurada.
- Marca visual del día de cierre semanal.
- Leyenda educativa para leer el calendario.
- Resumen semanal básico por perfil.
- Tareas manuales por día.
- Mover tareas manuales de un día a otro.
- Completar o reabrir tareas manuales.
- Guarda tareas en `profileWeeklyTasks`.
- Registra actividad en `activityLog`.

## Instalación

Copia estos archivos al repositorio `the86club-workspace` y reemplaza:

- index.html
- styles.css
- app.js
- firebase.js
- README.md
- assets
